นักศึกษาคณะ ICT

"มุ่งมั่นพัฒนาเว็บแอปพลิเคชันด้วย Django เพื่อสร้างโซลูชันที่มีประสิทธิภาพสำหรับสังคม"

ข้อมูลรายวิชา

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา ICT12367 - Django Web Application ซึ่งมุ่งเน้นการสร้างระบบจัดการฐานข้อมูลประชากรที่ใช้งานง่ายและมีความปลอดภัย

Backend

พัฒนาด้วย Python Django Framework เชื่อมต่อกับฐานข้อมูล SQLite

Frontend

ออกแบบด้วย HTML5, CSS3 และ Bootstrap 5.3 เพื่อการแสดงผลที่สวยงามทุกอุปกรณ์

หากมีข้อสงสัยหรือคำแนะนำเพิ่มเติม สามารถติดต่อผู้พัฒนาได้โดยตรงผ่านช่องทาง Social Media
